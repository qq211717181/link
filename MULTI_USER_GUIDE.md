# iLinks 多用户版使用指南

恭喜！您的 iLinks 导航站现在已经升级为支持多用户、数据库存储的完整版本。

## 🚀 快速开始

### 1. 启动服务
您需要同时启动前端和后端服务。

**后端服务 (API & 数据库):**
```bash
cd server
node server.js
```
*后端运行在 http://localhost:3001*

**前端服务 (界面):**
```bash
npm run dev
```
*前端运行在 http://localhost:5173*

### 2. 注册账号
1. 打开浏览器访问 `http://localhost:5173`
2. 点击右上角的 **"登录/注册"** 按钮
3. 点击底部的 **"立即注册"**
4. 输入用户名和密码进行注册

### 3. 导入书签
1. 注册/登录成功后，会自动跳转到 **后台管理** 页面
2. 在"书签导入"区域，点击上传按钮
3. 选择您从 Chrome 或 Edge 导出的书签 HTML 文件
4. 系统会自动解析并保存到数据库

### 4. 查看主页
1. 点击顶部的 **"返回首页"** 按钮
2. 您将看到您刚刚导入的书签分类
3. 您可以拖拽调整分类的顺序（修改会自动保存）

## ✨ 新特性

- **多用户支持**：不同用户拥有独立的书签数据，互不干扰
- **数据持久化**：使用 SQLite 数据库保存数据，不再依赖浏览器缓存
- **后台管理**：专属的管理页面，方便导入和管理数据
- **安全认证**：使用 JWT 进行用户认证，保护您的数据安全

## 🛠️ 技术栈

- **前端**：React, TypeScript, Tailwind CSS, Shadcn UI
- **后端**：Node.js, Express
- **数据库**：SQLite3
- **认证**：JWT (JSON Web Tokens)
