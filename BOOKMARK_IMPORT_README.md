# 谷歌书签导入功能

## 功能概述

已成功为您的 iLinks 项目添加了谷歌书签导入功能！现在您可以直接导入 Chrome/Edge 浏览器的书签文件，每个书签文件夹会自动转换为一个分类卡片。

## 新增功能

### 1. 导入书签
- 📤 **一键导入**: 点击"导入谷歌书签"按钮，选择浏览器导出的 HTML 书签文件
- 📁 **自动分类**: 每个书签文件夹自动转换为一个分类卡片
- 🔗 **保留链接**: 所有书签链接都会被保留，包括标题和 URL
- 💾 **本地存储**: 导入的书签保存在浏览器的 localStorage 中

### 2. 清除书签
- 🗑️ **一键清除**: 点击"清除导入的书签"按钮可以删除所有导入的书签
- ⚠️ **确认对话框**: 清除前会弹出确认对话框，防止误操作

### 3. 书签管理
- ➕ **添加链接**: 每个分类卡片都支持手动添加新链接
- 🗑️ **删除链接**: 鼠标悬停在链接上可以删除该链接
- 📂 **折叠/展开**: 支持折叠和展开分类卡片

## 使用步骤

### 第一步：导出浏览器书签

#### Chrome 浏览器
1. 按 `Ctrl + Shift + O` 打开书签管理器
2. 或者访问 `chrome://bookmarks/`
3. 点击右上角的三个点 `⋮` 菜单
4. 选择"导出书签"
5. 保存 HTML 文件到本地

#### Edge 浏览器
1. 按 `Ctrl + Shift + O` 打开收藏夹管理器
2. 或者访问 `edge://favorites/`
3. 点击右上角的三个点 `⋮` 菜单
4. 选择"导出收藏夹"
5. 保存 HTML 文件到本地

### 第二步：导入到 iLinks

1. 打开 iLinks 网站
2. 在搜索栏下方找到"导入谷歌书签"按钮
3. 点击按钮，选择刚才导出的 HTML 文件
4. 等待导入完成，会显示成功提示
5. 导入的书签文件夹会显示在页面上

## 技术实现

### 文件结构
```
src/
├── components/
│   └── BookmarkImporter.tsx  # 书签导入组件
└── pages/
    └── Index.tsx              # 主页面（已更新）
```

### 核心功能

#### BookmarkImporter 组件
- **HTML 解析**: 使用 DOMParser 解析 Chrome 书签 HTML 文件
- **递归处理**: 支持嵌套文件夹结构，子文件夹会被展平
- **错误处理**: 完善的错误提示和异常处理
- **UI 组件**: 使用 shadcn/ui 的 Button、AlertDialog 等组件

#### 数据持久化
- 使用 `localStorage` 存储导入的书签
- 页面刷新后自动恢复导入的书签
- 支持清除所有导入的书签

### 书签文件格式

Chrome/Edge 导出的书签文件是标准的 HTML 格式：

```html
<!DOCTYPE NETSCAPE-Bookmark-file-1>
<META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=UTF-8">
<TITLE>Bookmarks</TITLE>
<H1>Bookmarks</H1>
<DL><p>
    <DT><H3>工作</H3>
    <DL><p>
        <DT><A HREF="https://github.com">GitHub</A>
        <DT><A HREF="https://stackoverflow.com">Stack Overflow</A>
    </DL><p>
    <DT><H3>娱乐</H3>
    <DL><p>
        <DT><A HREF="https://youtube.com">YouTube</A>
        <DT><A HREF="https://netflix.com">Netflix</A>
    </DL><p>
</DL><p>
```

## 特性说明

### ✅ 支持的功能
- 导入 Chrome/Edge 书签 HTML 文件
- 自动解析文件夹和链接
- 支持嵌套文件夹（会被展平为多个分类）
- 本地存储导入的书签
- 清除所有导入的书签
- 手动添加/删除链接
- 折叠/展开分类

### ⚠️ 注意事项
- 只有包含链接的文件夹才会被导入
- 空文件夹会被自动忽略
- 导入的书签保存在浏览器 localStorage 中
- 清除浏览器数据会删除导入的书签
- 子文件夹会被展平，每个子文件夹都会成为独立的分类卡片

### 🎨 UI 设计
- 使用 glassmorphism（玻璃态）设计风格
- 深色主题，与现有设计保持一致
- 文件夹图标使用 `FolderOpen` 图标
- 导入按钮使用 `Upload` 图标
- 清除按钮使用红色主题的 `Trash2` 图标

## 问题排查

如果导入失败，请检查：

1. ✅ 文件格式是否正确（必须是 HTML 文件）
2. ✅ 文件是否是从 Chrome/Edge 导出的
3. ✅ 文件中是否包含有效的书签文件夹
4. ✅ 浏览器控制台是否有错误信息
